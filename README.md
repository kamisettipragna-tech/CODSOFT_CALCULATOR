# 🧮 Professional Calculator Web App

A modern, fully functional calculator web application built with Python Flask and vanilla JavaScript. This project was developed as part of my internship at **CodSoft**.

![Status](https://img.shields.io/badge/status-completed-success)

---

## ✨ Features

- ✅ **Basic Operations** - Addition, Subtraction, Multiplication, Division
- ✅ **Real-time Calculation** - Instant results as you type
- ✅ **Beautiful UI** - Modern glass-morphism design with dark theme
- ✅ **History Tracking** - Stores last 10 calculations
- ✅ **Keyboard Support** - Use your keyboard for faster input
- ✅ **Error Handling** - Graceful handling of division by zero and invalid inputs
- ✅ **Responsive Design** - Works on desktop, tablet, and mobile
- ✅ **Local Storage** - History persists even after page refresh
